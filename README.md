# Fisher cipher


Шифратор «Рыбачьей тайнописью» (по мотивам известной детской повести «Гигантия» Томаса Тейлора)

**Суть метода:**

Ключом служит одно из восьми направлений света (С, Ю, З, В, СЗ, СВ, ЮЗ, ЮВ).
При шифровании из стандартного начертания букв удаляются все линии, ориентированные в направлении ключа. В точке начала каждой удалённой линии ставится пометка (жирная точка).

**Процесс расшифровки:**

Получатель знает ключевое направление («СЕВЕР») и зашифрованное сообщение.
Из каждой жирной точки получатель проводит линию в заданном направлении.

В произведении «Гигантия» направление-ключ печаталось отдельно в виде компаса, разделенного на две части. Читатель накладывал части друг на друга, чтобы узнать сторону света.
В реализации алгоритма сохранен этот принцип: ключ передается вместе с посланием, но для расшифровки достаточно знать само направление.

Начертание букв реализатором алгоритма приведено к единому печатному стилю, для адаптации под цифровое использование.
